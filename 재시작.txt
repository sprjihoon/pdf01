프로그램 재시작 방법
================================================================

현재 실행 중인 GUI 창을 닫고 다시 실행하세요.

[방법 1] 바로가기 파일 사용
  → PDF정렬_실행.bat 더블클릭!

[방법 2] 명령 프롬프트
  → python main.py

[방법 3] 파일 탐색기
  → main.py 파일 더블클릭!
